apidoc supports the notion of 'attributes' that can be used to enhance code generation:

  • Attributes are globally unique and have URL friendly names.
  • Attributes have an optional description.
  • Attribute values can be set at the organization level, cascading to all of the applications that belong to that organization.
  • A code generator can return a list of the attribute names on which it depends. apidoc will then send any matching attribute values when invoking the code generator.
